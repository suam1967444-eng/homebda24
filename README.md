## 4. Project ①  
### 🛒 E-commerce Customer Churn Analysis

**Goal**  
다음 달 이탈 가능성이 높은 고객을 예측하고 매출 손실을 줄인다.

**Data**
- 고객 20만 명
- 주문 150만 건 (Users, Orders, Payments)

**Workflow**
- SQL: 고객별 구매 빈도·최근성 집계
- Excel: 구매 횟수별 이탈률 빠른 검증
- Python: 이탈 예측 모델(Logistic / RandomForest)
- Tableau: 이탈 위험 고객 대시보드

**Key Insight**
- 첫 구매 후 14일 무구매 시 이탈 확률 급증
- 구매 금액보다 구매 빈도가 핵심 변수

**Action**
- 10일 무구매 고객 자동 쿠폰 타겟팅
- 신규 고객 온보딩 메시지 개선

## 5. Project ②  
### 🏠 Real Estate Price Signal Detection

**Question**  
가격 상승 전에 나타나는 거래 신호는 무엇인가?

**Data**
- 아파트 실거래 30만 건
- 지역, 층, 면적, 거래일자

**Analysis**
- SQL: 지역별 거래량·평단가 집계
- Python: 거래량-가격 시차 분석
- Tableau: 지역별 가격 변화 지도

**Finding**
- 거래량 반등 후 1~2분기 뒤 가격 상승
- 저층 거래 회복이 초기 상승 신호로 작용

## 6. Project ③  
### 📱 User Behavior Funnel Analysis

**Objective**
가입 후 3일 이내 이탈 원인 분석

**Analysis**
- 퍼널 분석(가입 → 첫 행동 → 반복 행동)
- 메시지 수·활동일수와 잔존율 비교

**Insight**
- 첫 3일 메시지 5회 미만 시 이탈 확률 2배

**Recommendation**
- 초기 3일 미션형 UX 도입
- 대화 유도 알림 설계
